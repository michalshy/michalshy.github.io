The plan is to keep posts short and focused — notes more than essays. Topics I'm likely to cover:

## Format

Articles are plain Markdown files. No CMS, no database, no build step. Just files.

Code blocks work as expected:

```rust

    let greeting = "hello from the first post";
    println!("{}", greeting);
}
```
